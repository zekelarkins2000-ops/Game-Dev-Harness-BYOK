# GBA RPG Prompt

Create a first playable for an original top-down monster-catching RPG for GBA.

Constraints:
- Use original region, original story, and original human character sprites.
- The only reusable concept should be generic monster-catching mechanics, not copyrighted assets.
- Use placeholder tiles and sprites.
- Start with one room, one NPC, one encounter, one menu, and one save-like state plan.
- Include devkitPro build instructions and QA checks.
